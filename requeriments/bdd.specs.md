
## Narrativa 

Como visitante quero me cadastrar no sistema para me tornar um cliente

## Cenário

Quando o visitante clicar em cadastrar o sistema deve oferecer um formulário para preenchimento
apos o preenchimento o visitante clica em ENVIAR é enviado um POST para o servidor onde os dados serão verificados e se tudo estiver correto então os dados serão inseridos no banco e deve ser enviado uma mensagem para o cliente, se os dados estiverem incorretos então retorna mensagem de erro para o cliente. 

## Narrativa
Como cliente quero fazer login no sistema

## Cenário

Quando o cliente acessar o sistema devera ser efetuado o login no sistema indormando email e senha o que vai gerar um token e disponibilizar acesso ao sistema.
